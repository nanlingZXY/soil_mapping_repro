## 二、项目结构
| 文件夹/文件 | 功能说明 |
| ---- | ---- |
| data/ | 原始数据集（本地存放，需网盘下载） |
| code/ | 核心代码文件，项目运行脚本 |
| results/ | 实验结果输出，保存图表/数据 |
| environment.yml | conda环境配置文件 |
| README.md | 项目说明文档 |

## 三、运行环境
- Python 3.9 及以上版本
- 操作系统：Windows 10/11
- 推荐使用 Conda 管理环境

## 五、使用说明
- 配置 Python/Conda 运行环境
- 下载数据并放置到指定目录
- 运行 code/ 文件夹内的代码文件
- 运行结果自动保存至 results/ 文件夹
